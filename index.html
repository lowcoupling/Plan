Plan

Plan is an open source project for Eclipse allowing you to easily define plans and schedules in the same way you code. Plan is a Domain Specific Language (DSL), specifically designed to represents aspects of project plans like activities, dependencies, checkpoints, etc.

The editor of the Plan language is based on XText and includes: *keyword highlights; *code completion; *tree baed outline;

Plan integrates the Nebula Gantt View allowing for a realtime graphical representation of the plan you are defining.
